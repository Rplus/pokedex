0. 共通

  1. 點擊屬性圖示 => 屬性相剋圖
  2. 有三角符號的標題可點擊展現更多內容，
    並且會記住開閤狀態
  3. 表格標頭有漸層者即為當前排序欄，
    點擊切換排序欄或方向
  4. 部份表格標頭滑鼠懸停可顯示標頭說明
  5. 招式與寶可夢同屬性時，字體樣式會加粗提示
  6. 下方的留言首次需點開才會載入

1. /Home

以 [關鍵字] 搜尋 [寶可夢] / [招式]

  • 寶可夢:
    • Pokedex(數字)
    • 名字(zh, en)
    • 屬性(en)
  • 招式:
    • 名字(zh, en)
    • 屬性(en)

2. /Move

招式列表

  • 屬性過濾
    • 點擊可單獨列出某一屬性的招式
    • 圖示下方中文字可取得該屬性連結 /move#fire
  • PVP / PVE 切換鈕
    切換該招式在各模式下的數值
  • 點擊招式名稱 => 招式頁

2. /Pokemon

寶可夢列表

  • 屬性過濾
    • 點擊可單獨列出某一屬性的寶可夢
    • 圖示下方中文字可取得該屬性連結 /pokemon#fire
  • 點擊寶可夢名稱 => 寶可夢頁
  • 寶可夢頁下方可模擬對戰方屬性，進而更新各招式數值
